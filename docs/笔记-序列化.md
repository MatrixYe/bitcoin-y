手写 Bitcoin Core 的共识序列化路径：
- 整数使用 little-endian
- hash 字段使用 internal byte order
- vector/脚本长度使用 CompactSize
- block header 固定 80 字节
- block = header + txn_count + transactions
- txid/block hash 基于这些共识字节做 double SHA256
